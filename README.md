# 火箭与第三帝国

[@李挺诺夫](http://weibo.com/kkndyu)      翻译连载中

本书译自《The Rocket and the Reich》，讲述了纳粹德国V-2火箭研发历程。火箭发动机、超音速空气动力、制导三大难题如何攻克，元首的赞赏如何转化为实际的资源，劳工匮乏的困境如何化解，党卫队如何介入佩内明德，这一切尽在《火箭和第三帝国》。

在线阅读：[GitBook](https://www.gitbook.com/read/book/kkndyu/rocket) 或 [Github](https://github.com/kkndyu/rocket/blob/master/SUMMARY.md)。
